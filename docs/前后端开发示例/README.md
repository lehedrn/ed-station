## 🛠️ SQL 脚本组织

建议将所有 SQL 脚本按以下结构组织：

```
ed-station-boot/sql/
├── 01_ddl/                    # DDL 脚本（按执行顺序编号）
│   ├── 001_blade_xxx.sql
│   └── ...
├── 02_dict/                   # 字典配置
│   ├── dict_xxx.sql
│   └── ...
├── 03_menu/                   # 菜单权限
│   ├── menu_xxx.sql
│   └── ...
└── README.md                  # SQL 脚本说明
```
