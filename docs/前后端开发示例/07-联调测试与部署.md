# 联调测试指南

## 一、联调前准备

### 1.1 执行 SQL 脚本

在联调前，先执行所有配置 SQL 脚本：

```bash
# 1. 执行建表脚本
mysql -u root -p your_database < sql/01_ddl/001_blade_inner_message.sql

# 2. 执行字典配置
mysql -u root -p your_database < sql/02_dict/dict_inner_message.sql

# 3. 执行菜单权限配置
mysql -u root -p your_database < sql/03_menu/menu_inner_message.sql
```

> **注意**：执行完成后需重启后端服务或刷新 Redis 缓存，使菜单和字典配置生效

### 1.2 刷新缓存

```bash
# 方式 1：重启后端服务
systemctl restart ed-station-boot

# 方式 2：Redis 命令行刷新
redis-cli
> FLUSHDB  # 谨慎使用，会清空当前数据库所有数据
```

### 1.3 确认服务启动

```bash
# 前端服务（端口 3888）
pnpm dev

# 后端服务（端口 18080）
mvn spring-boot:run
```

## 二、后端接口测试

### 2.1 访问 Swagger 文档

浏览器访问：http://localhost:18080/doc.html

### 2.2 测试顺序

1. **新增接口** (`/blade-desk/xxx/save` 或 `/submit`)
   - 测试新增一条数据
   - 验证必填字段校验
   - 验证数据正确入库

2. **分页查询接口** (`/blade-desk/xxx/list`)
   - 测试查询列表
   - 测试分页参数
   - 测试搜索条件

3. **详情接口** (`/blade-desk/xxx/detail`)
   - 测试查询详情
   - 验证字典值转换

4. **修改接口** (`/blade-desk/xxx/update` 或 `/submit`)
   - 测试修改数据
   - 验证修改后数据正确

5. **删除接口** (`/blade-desk/xxx/remove`)
   - 测试单条删除
   - 测试批量删除

### 5.2 常见问题及解决方案

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 401 未授权 | Token 过期 | 重新登录 |
| 403 无权限 | 权限未配置 | 检查菜单权限配置 |
| 字典不显示 | Redis 缓存未刷新 | 重启后端或刷新 Redis |
| 数据为空 | 查询条件问题 | 检查查询参数 |
| 500 错误 | 后端异常 | 查看后端日志 |
