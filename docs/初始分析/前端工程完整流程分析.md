## 二、登录流程详解

### 1. 登录页面入口

**文件**: `src/page/login/userlogin.vue`

```
用户输入 → 表单验证 → handleLogin() → dispatch('LoginByUsername')
```

**关键步骤**:

1. **初始化** (`created` 钩子):
   - `getTenant()` - 获取租户信息（调用 `/blade-system/tenant/info`）
   - `refreshCode()` - 获取验证码（调用 `/blade-auth/oauth/captcha`）

2. **默认值**:
   ```javascript
   tenantId: '000000'
   username: 'admin'
   password: 'admin'
   type: 'account'
   ```

### 2. 登录 API 调用

**文件**: `src/api/user.js`

```javascript
loginByUsername(tenantId, deptId, roleId, username, password, type, key, code)
  → POST /blade-auth/oauth/token
```

**请求头**:
- `Tenant-Id`: 租户 ID
- `Dept-Id`: 部门 ID
- `Role-Id`: 角色 ID
- `Captcha-Key`: 验证码 Key
- `Captcha-Code`: 验证码值

**参数**:
- `grant_type`: 'captcha' 或 'password'
- `scope`: 'all'

### 3. 密码加密

**文件**: `src/utils/sm2.js`

登录前使用国密 SM2 算法对密码进行加密:
```javascript
encrypt(userInfo.password)
```

### 4. Store 处理登录响应

**文件**: `src/store/modules/user.js`

**Action: `LoginByUsername`**:
```javascript
1. 调用 loginByUsername API
2. 成功后 Commit:
   - SET_TOKEN: access_token
   - SET_REFRESH_TOKEN: refresh_token
   - SET_TENANT_ID: tenant_id
   - SET_USER_INFO: 用户信息
   - DEL_ALL_TAG: 清空标签
   - CLEAR_LOCK: 清除锁屏
```

### 5. 加载工作流路由

登录成功后调用:
```javascript
dispatch('FlowRoutes')
```

### 6. 跳转到首页

```javascript
this.$router.push(this.tagWel)  // tagWel 为欢迎页路由
```

## 三、路由守卫与权限控制

### 1. 路由守卫

**文件**: `src/permission.js`

```javascript
router.beforeEach((to, from, next) => {
  // 1. 检查 Token
  if (getToken()) {
    // 已登录
    if (store.getters.isLock && to.path !== '/lock') {
      // 锁屏状态跳转到锁屏页
      next({ path: '/lock' });
    } else if (to.path === '/login') {
      // 已登录访问登录页跳转到主页
      next({ path: '/' });
    } else {
      // 检查 token 是否为空
      if (store.getters.token.length === 0) {
        dispatch('FedLogOut').then(() => next({ path: '/login' }));
      } else {
        // 添加标签页
        if (meta.isTab !== false) {
          commit('ADD_TAG', { ... });
        }
        next();
      }
    }
  } else {
    // 未登录
    if (meta.isAuth === false) {
      next();  // 允许访问
    } else {
      next('/login');  // 跳转到登录页
    }
  }
});
```

**openMenu 方法**:
```javascript
openMenu(item = {}) {
  // 1. 调用 GetMenu Action 获取菜单
  this.$store.dispatch('GetMenu', item.id).then(data => {
    // 2. 格式化路由
    this.$router.$avueRouter.formatRoutes(data, true);
    // 3. 跳转到默认路径
    if (!validatenull(item.path)) {
      this.$router.push({ path: item.path });
    }
  });
}
```

### 4. 获取菜单数据

**文件**: `src/store/modules/user.js`

```javascript
GetMenu({ commit, dispatch }, topMenuId) {
  getRoutes(topMenuId)  // GET /blade-system/menu/routes
    .then(res => {
      const data = res.data.data;
      let menu = deepClone(data);
      menu.forEach(ele => formatPath(ele, true));
      commit('SET_MENU', menu);
      commit('SET_MENU_ALL', menu);
      dispatch('GetButtons');  // 获取按钮权限
      resolve(menu);
    });
}
```

### 5. 获取按钮权限

```javascript
GetButtons({ commit }) {

    .then(res => {
      const data = res.data.data;
      commit('SET_PERMISSION', data);
      // 解析权限码为对象 { code: true }
    });
}
```

## 六、HTTP 请求拦截处理

**文件**: `src/axios.js`

### 请求拦截器
```javascript
axios.interceptors.request.use(config => {
  NProgress.start();  // 进度条
  
  // URL 处理
  config.url = baseUrl + config.url;
  
  // 添加认证头
  config.headers['Authorization'] = `Basic ${Base64.encode(clientId:clientSecret)}`;
  
  // 添加 Token
  config.headers[website.tokenHeader] = 'bearer ' + getToken();
  
  // 支持报文加密
  if (config.cryptoData === true) {
    config.data = crypto.encryptAES(JSON.stringify(config.data), crypto.aesKey);
  }
  
  return config;
});
```

### 响应拦截器
```javascript
axios.interceptors.response.use(res => {
  NProgress.done();
  
  // 解密响应数据
  if (config.cryptoData === true) {
    res.data = JSON.parse(crypto.decryptAES(res.data, crypto.aesKey));
  }
  
  // 状态码处理
  if (status === 401) {
    // 尝试刷新 token
    if (!config._retry) {
      config._retry = true;
      return store.dispatch('RefreshToken').then(() => axios(config));
    } else {
      // 刷新失败，跳转登录
      store.dispatch('FedLogOut').then(() => router.push('/login'));
    }
  }
  
  if (status !== 200) {
    ElMessage.error(res.data.msg);
    return Promise.reject();
  }
  
  return res;
});
```

## 十一、安全特性

1. **国密 SM2 加密**: 密码传输前使用 SM2 加密
2. **Token 机制**: JWT Bearer Token 认证
3. **双 Token**: access_token + refresh_token
4. **自动刷新**: Token 过期自动刷新机制
5. **权限控制**: 路由级 + 按钮级权限控制
6. **多租户**: 租户隔离支持
7. **多角色**: 支持多角色切换
8. **多部门**: 支持多部门切换
