## 技术栈

### 核心框架
- **Spring Boot 3.2.10**
- **BladeX 4.8.0** (blade-bom 统一管理)
- **MyBatis-Plus** - ORM
- **Druid** - 连接池

### 安全认证
- **blade-core-oauth2** - OAuth2
- **blade-starter-tenant** - 多租户 (字段隔离)
- **blade-starter-api-crypto** - 接口加密
- **blade-starter-data-scope** - 数据权限
- **blade-starter-data-record** - 操作日志

### 存储
- **MySQL** (主支持)
- **Redis** - 缓存/Session
- **Ehcache** - 本地缓存
- **OSS/MinIO/S3** - 对象存储 (多服务商支持)

### 功能模块
| 模块 | 依赖 |
|------|------|
| 任务调度 | blade-starter-powerjob |
| 接口文档 | knife4j-openapi3-ui |
| Excel | blade-starter-excel |
| 报表 | blade-starter-report |
| 验证码 | easy-captcha |
| 国际化和 | blade-starter-i18n |
| 代码生成 | mybatis-plus-generator |

### 短信/存储服务商
- **OSS**: MinIO, 阿里云，腾讯云，华为云，七牛，AWS S3
- **SMS**: 阿里云，腾讯云，云片

## 模块说明

### common
通用工具、配置、常量、缓存、事件、处理器

### modules/auth
OAuth2 认证、授权、Token 管理、第三方登录

### modules/system
- 用户、角色、菜单、部门
- 字典、参数、岗位
- 租户管理
- 数据权限
- 操作日志
- 规则引擎 (rule/)

### modules/desk
工作台、仪表盘、通知公告

### modules/resource
文件上传、OSS、SMS

### job
PowerJob 集成、定时任务

## 开发命令

```bash
# 创建日志目录
mkdir -p /home/workspace/com/ed-station/logs

# 启动应用 (端口 18080)
mvn spring-boot:run > /home/workspace/com/ed-station/logs/backend.log 2>&1 &

# 打包
mvn package             # 打包
mvn package -DskipTests # 跳过测试

# 查看日志
tail -f /home/workspace/com/ed-station/logs/backend.log
```
