# ed-station 项目架构分析

> 更新时间：2026-04-01

## 项目概述

**ed-station** 是基于 BladeX 4.8.0 的企业级中后台系统，采用前后端分离架构。

## 项目结构

```
ed-station/
├── ed-station-web/     # Vue 3 前端
├── ed-station-boot/    # Spring Boot 后端
└── docs/               # 文档
```

## 架构图

```
┌─────────────────────────────────────────┐
│              前端层                       │
│  Vue 3 + Element Plus + Avue            │
│  (端口 3888)                             │
│  - 支持 macOS 风格界面适配                │
└─────────────────────────────────────────┘
                    │
                    │ HTTP/JSON
                    ▼
┌─────────────────────────────────────────┐
│              应用层                       │
│  Spring Boot + BladeX                   │
│  (端口 18080)                            │
└─────────────────────────────────────────┘
        │               │       │
        ▼               ▼       ▼
   ┌────────┐    ┌────────┐ ┌──────┐
   │ MySQL  │    │ Redis  │ │ OSS  │
   └────────┘    └────────┘ └──────┘
```

## 功能模块

### 系统管理
- 用户、角色、菜单、部门
- 字典、参数、岗位
- 租户管理

### 认证授权
- OAuth2 认证
- JWT Token
- 第三方登录 (GitHub/Gitee/微信/QQ/钉钉)

### 多租户
- 租户管理
- 租户套餐
- 字段隔离 (tenant_id)

### 资源管理
- 文件上传
- 对象存储 (多云支持)
- 短信服务

### 任务调度
- PowerJob 集成
- 定时任务

### 工作台
- 仪表盘、通知公告

- Node.js 18+

- Redis 5.0+

## 开发流程

### 创建日志目录
```bash
mkdir -p /home/workspace/com/ed-station/logs
```

### 启动后端
```bash
cd ed-station-boot
mvn spring-boot:run > /home/workspace/com/ed-station/logs/backend.log 2>&1 &
```

### 启动前端
```bash
cd ed-station-web
pnpm dev > /home/workspace/com/ed-station/logs/frontend.log 2>&1 &
```

### 查看日志
```bash
tail -f /home/workspace/com/ed-station/logs/frontend.log  # 前端日志
tail -f /home/workspace/com/ed-station/logs/backend.log   # 后端日志
```

### 访问
http://localhost:3888

前端自动代理 `/api` 到 `http://localhost:18080`

## 核心特性

| 特性 | 说明 |
|------|------|
| OAuth2 | 标准认证 |
| JWT | 无状态令牌 |
| 国密 | SM2/SM3/SM4 |
| XSS 防护 | 自动过滤 |
| 数据权限 | 行级控制 |
| 接口加密 | AES/DES |
| 操作日志 | 自动记录 |
| 多租户 | 字段隔离 |
| macOS 适配 | 苹果系统界面适配 |

## 数据库表前缀

- `blade_` - 系统表
- `blade_user` - 用户表 (多租户排除)
