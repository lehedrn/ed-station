- **Element Plus 2.10.1** + `@element-plus/icons-vue 2.3.1`
- **@smallwei/avue 3.7.2** - 低代码中后台框架

## 配置

### Vite (vite.config.mjs)
- **开发端口**: 3888
- **API 代理**: `/api` → `http://localhost:18080`
- **路径别名**:
  - `@` → `src/`
  - `components` → `src/components`
  - `styles` → `src/styles`
  - `utils` → `src/utils`
- **SCSS 全局变量**: `variables.scss`

### 构建
- **目标**: esnext
- **生产压缩**: Terser (移除 console/debugger)
- **代码分割**: element-plus, @smallwei/avue

## 核心文件

### src/main.js
应用入口，注册：
- Element Plus 图标
- 全局组件 (codeEditor, cronEditor, flowDesign 等)
- 全局属性 ($app, $dayjs, website)

### src/axios.js
HTTP 请求封装，含国密加密逻辑

### src/permission.js
路由守卫和权限验证

### src/mixins/crud.js
通用 CRUD 操作混入

### src/mac/
macOS 风格界面适配
- `login.vue` - macOS 风格登录页
- `lock.vue` - 锁屏界面
- `index.vue` - macOS 风格主界面

## 开发命令

```bash
# 创建日志目录
mkdir -p /home/workspace/com/ed-station/logs

# 启动开发服务器 (端口 3888)
pnpm dev > /home/workspace/com/ed-station/logs/frontend.log 2>&1 &

# 构建
pnpm build        # 构建 (开发环境)
pnpm build:prod   # 构建 (生产环境)
pnpm serve        # 预览构建结果

# 查看日志
tail -f /home/workspace/com/ed-station/logs/frontend.log
```
