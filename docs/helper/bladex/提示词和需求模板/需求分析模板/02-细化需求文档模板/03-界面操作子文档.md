## 1. 界面设计总则

### 1.1 设计原则

| 原则 | 说明 |
|------|------|
| 一致性 | 同类操作使用一致的交互方式，降低用户学习成本 |
| 易用性 | 符合用户操作习惯，减少不必要的点击和跳转 |
| 反馈性 | 操作后及时给予明确反馈，告知用户当前状态 |
| 容错性 | 关键操作需确认，支持撤销或回退 |
| 效率性 | 常用功能便捷可达，支持快捷键和批量操作 |

### 1.2 技术规范

| 端 | UI 框架 | 组件库 |
|---|--------|--------|
| PC 端 | Vue 3 (Options API) | Element Plus + Avue 3.7 |
| 移动端 | UniApp (Vue 3) | uView UI 2.x |

### 1.3 响应式设计

| 端 | 最小宽度 | 推荐分辨率 |
|---|---------|-----------|
| PC 端 | 1280px | 1920×1080 |
| 移动端 | 320px | 375×667 (iPhone SE) |

## 6. BladeX 前端规范

### 6.1 API 请求封装

所有接口请求使用统一的 axios 封装：

```javascript
import request from '@/axios';

// 列表示例
export const getList = (current, size, params) => {
  return request({
    url: '/blade-[模块]/[资源]/list',
    method: 'get',
    params: { ...params, current, size },
    cryptoToken: false,
    cryptoData: false,
  });
};
```

### 6.2 字典使用

字典数据通过 BladeX 统一字典接口获取：

```javascript
// Avue 配置示例
{
  label: '[字段标签]',
  type: 'select',
  dicUrl: '/blade-system/dict/dictionary?code=[dict_code]',
  props: { label: 'dictValue', value: 'dictKey' },
  dataType: 'number', // 或 'string'
  prop: '[字段名]',
}
```

### 6.3 权限控制

按钮权限通过 `permission` 对象控制：

```vue
<el-button 
  v-if="permission['module_add']" 
  type="primary" 
  @click="handleAdd"
>
  新增
</el-button>
```

### 6.4 日期格式

- 列表展示：`YYYY-MM-DD HH:mm:ss`
- 日期选择：`YYYY-MM-DD`
- 时间选择：`HH:mm:ss`
- 日期时间：`YYYY-MM-DD HH:mm:ss`

使用 Dayjs 进行日期格式化。
