## 可能的问题和解决方案

### 1. 路由问题
如果前端页面无法访问，检查：
- 后端菜单配置是否正确（`blade_menu` 表）
- 前端路由是否注册

### 2. 接口 404
如果接口返回 404，检查：
- 后端服务是否启动
- 模块是否加载（检查 `pom.xml` 依赖）
- 接口路径是否正确

### 3. 权限问题
如果按钮不显示，检查：
- 用户是否有对应角色
- 菜单权限是否配置（`blade_menu` 和 `blade_role_menu` 表）

### 4. 字典数据问题
如果字典值不显示，检查：
- 字典是否已配置（`blade_dict` 表）
- 字典接口是否可访问
