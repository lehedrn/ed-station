## 1. 字典体系概述

### 1.1 字典表结构

BladeX 字典使用 `blade_dict` 和 `blade_dict_data` 两张表：

- `blade_dict`：字典类型表（定义字典类别）
- `blade_dict_data`：字典数据表（定义具体字典项）

### 1.2 本系统字典清单

| 字典编码 | 字典名称 | 用途 | 数据项数 |
|---------|---------|------|---------|
| affair_type | 事项类别 | 事项分类 | 6 项 |
| material_type | 材料类型 | 材料分类 | 4 项 |
| affair_status | 事项状态 | 事项状态 | 3 项 |

### 4.4 使用 BladeX 字典工具类

```javascript
// src/utils/dict.js
import request from '@/axios';

// 字典缓存
const dictCache = new Map();

/**
 * 获取字典数据
 * @param {string} dictCode 字典编码
 * @returns {Promise<Array>} 字典数据数组
 */
export const getDict = async (dictCode) => {
  if (dictCache.has(dictCode)) {
    return dictCache.get(dictCode);
  }
  
  const response = await request({
    url: '/blade-system/dict/dictionary',
    method: 'get',
    params: { code: dictCode },
    cryptoToken: false,
    cryptoData: false,
  });
  
  const dictData = response.data || [];
  dictCache.set(dictCode, dictData);
  return dictData;
};

/**
 * 获取字典标签
 * @param {string} dictCode 字典编码
 * @param {string|number} dictKey 字典键值
 * @returns {string} 字典标签
 */
export const getDictLabel = async (dictCode, dictKey) => {
  const dictData = await getDict(dictCode);
  const item = dictData.find(d => d.dictKey === String(dictKey));
  return item ? item.dictValue : '';
};

/**
 * 同步获取字典标签（需要先缓存）
 */
export const getDictData = (dictCode, dictKey) => {
  const dictData = dictCache.get(dictCode) || [];
  const item = dictData.find(d => d.dictKey === String(dictKey));
  return item ? item.dictValue : '';
};
```

### 5.3 Service 层字典值填充

```java
// AffairServiceImpl.java
@Override
public IPage<AffairVO> pageList(Page page, Map<String, Object> params) {
    // 查询列表
    IPage<Affair> affairPage = baseMapper.selectPage(page, getQueryWrapper(params));
    
    // 转换为 VO 并填充字典值
    List<AffairVO> voList = affairPage.getRecords().stream()
        .map(affair -> {
            AffairVO vo = BeanUtil.copyProperties(affair, AffairVO.class);
            
            // 填充事项类别字典值
            vo.setAffairTypeDict(getDictValue("affair_type", affair.getAffairType()));
            
            // 填充状态字典值
            vo.setStatusDict(getDictValue("affair_status", String.valueOf(affair.getStatus())));
            
            return vo;
        })
        .collect(Collectors.toList());
    
    IPage<AffairVO> voPage = new Page<>(affairPage.getCurrent(), affairPage.getSize(), affairPage.getTotal());
    voPage.setRecords(voList);
    return voPage;
}

/**
 * 获取字典值
 */
private String getDictValue(String dictCode, String dictKey) {
    List<DictDataVO> dictList = SpringUtil.getBean(IDictService.class)
        .getList(dictCode);
    if (dictList != null) {
        Optional<DictDataVO> optional = dictList.stream()
            .filter(d -> d.getDictKey().equals(dictKey))
            .findFirst();
        return optional.map(DictDataVO::getDictValue).orElse("");
    }
    return "";
}
```

## 7. 注意事项

### 7.1 字典数据缓存

- BladeX 字典数据会自动缓存到 Redis
- 修改字典后需要清除缓存或重启服务
- 前端字典数据建议缓存在 Vuex 中

### 7.2 字典删除限制

- 已被业务数据引用的字典项不建议删除
- 如需停用，建议通过状态字段控制
- 删除前需校验是否有数据引用

### 7.3 字典扩展

- 后续如需新增字典类型，按此规范扩展
- 字典编码命名规范：`模块_类型`，如 `affair_type`
- 字典项 key 值建议使用数字或两位编码

### 7.4 字典与枚举的对应关系

后端代码中建议同时维护枚举类，便于类型安全：

```java
// AffairType.java
@Getter
@AllArgsConstructor
public enum AffairType {
    
    ADMINISTRATIVE_LICENSE("01", "行政许可"),
    ADMINISTRATIVE_CONFIRMATION("02", "行政确认"),
    ADMINISTRATIVE_RULING("03", "行政裁决"),
    ADMINISTRATIVE_GIVING("04", "行政给付"),
    PUBLIC_SERVICE("05", "公共服务"),
    OTHER("06", "其他");
    
    private final String code;
    private final String name;
    
    public static String getNameByCode(String code) {
        for (AffairType type : values()) {
            if (type.getCode().equals(code)) {
                return type.getName();
            }
        }
        return "";
    }
}
```

## 9. 附录：字典查询 SQL 参考

```sql
-- 查询事项类别字典
SELECT d.dict_key, d.dict_value, d.sort
FROM blade_dict_data d
JOIN blade_dict t ON d.dict_id = t.id
WHERE t.code = 'affair_type'
ORDER BY d.sort;

-- 查询材料类型字典
SELECT d.dict_key, d.dict_value, d.sort
FROM blade_dict_data d
JOIN blade_dict t ON d.dict_id = t.id
WHERE t.code = 'material_type'
ORDER BY d.sort;

-- 查询事项状态字典
SELECT d.dict_key, d.dict_value, d.sort
FROM blade_dict_data d
JOIN blade_dict t ON d.dict_id = t.id
```
